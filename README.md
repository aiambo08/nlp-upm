# Trabajos de Procesamiento del Lenguaje Natural (UPM)

Este repositorio contiene dos prácticas desarrolladas en la asignatura de **Procesamiento del Lenguaje Natural** del Grado en Ciencia de Datos e Inteligencia Artificial (UPM):

- `1_clasificacion_grupo_3.ipynb`: Clasificación de textos usando modelos de Machine Learning.
- `3_extraccion_grupo_3.ipynb`: Extracción de información / entidades a partir de texto.

Cada notebook incluye:
- Preprocesamiento de texto.
- Entrenamiento y evaluación de modelos.
- Visualización e interpretación de resultados.
